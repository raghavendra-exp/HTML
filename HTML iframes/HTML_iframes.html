HTML iframes

An iframe (inline frame) is used to display a web page within a web page. The iframe element is often used to embed content from another source, such as a video, map, or another website.

Attributes of the iframe element

  • src: Specifies the URL of the page to embed.
  • width: Specifies the width of the iframe.
  • height: Specifies the height of the iframe.
  • frameborder: Specifies whether to display a border around the iframe (values: "0" or "1").
  • title: Provides a title for the iframe for accessibility purposes.

Note

Some websites may have restrictions that prevent them from being displayed in an iframe on other domains (this is often done for security reasons).
